# 기존 GitHub 저장소 업데이트

1. 이 ZIP을 압축 해제합니다.
2. GitHub의 lucky-lab 저장소에서 Add file > Upload files를 누릅니다.
3. 이 폴더 안의 `public`, `functions`, `docs`, `package.json`, `wrangler.toml`, `README.md`를 모두 업로드합니다.
4. 같은 이름의 파일은 교체됩니다.
5. Commit changes를 누릅니다.
6. Cloudflare Pages가 자동으로 새 배포를 시작합니다.
7. Deployments에서 초록색 체크를 확인합니다.
