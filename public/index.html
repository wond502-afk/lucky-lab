최신 당첨번호 확인

회차별 당첨번호 조회

FAQ

자주 묻는 질문

번호 생성은 완전한 무작위인가요?

기본적으로 무작위이며, 선택한 홀짝·구간·끝수 조건을 만족하는 조합을 생성합니다.

저장한 번호는 어디에 저장되나요?

현재 사용 중인 브라우저의 로컬 저장소에만 저장되며 별도 서버로 전송되지 않습니다.

많이 나온 번호가 다음에도 잘 나오나요?

아닙니다. 각 추첨은 독립적이며 과거 출현 빈도가 다음 추첨 확률을 높이지 않습니다.

생성된 번호가 당첨을 보장하나요?

보장하지 않습니다. 모든 번호 조합의 1등 당첨 확률은 같습니다.
